## T-5: hub-client SDK（Node.js + Python）

**时间**：2026-06-05

**任务**：为 agent-hub 平台创建 Node.js 和 Python 双语言客户端 SDK，供业务项目集成。

**执行内容**：

1. **Node.js SDK** (`sdk/hub-client.js`, 296 行)
   - 纯 CommonJS 模块，零外部依赖（仅 http/https/crypto/url 内置模块）
   - 类 `HubClient`，构造函数接受 `{ baseUrl, apiKey, businessCode, token }`
   - 内部 `_request(method, path, body)` 统一处理：HTTPS 请求、JSON 序列化、双认证头注入、响应 `data` 解包
   - 内部 `_generateToken()` 使用 `crypto.randomUUID()` 生成 UUID v4
   - 所有 worker 端方法：`heartbeat`、`acquireLock`/`renewLock`/`releaseLock`、`createPlaybook`/`searchPlaybooks`/`getPlaybook`、`appendEvent`、`getBusinessByCode`
   - 409 锁冲突抛出 `err.code === 409` 的 Error，附带 `response` 属性
   - 全部方法含 JSDoc 注释

2. **Python SDK** (`sdk/hub_client.py`, 312 行)
   - 纯标准库实现（urllib.request + json + uuid），兼容 Python 3.8+
   - 类 `HubClient`，构造函数接受 `(base_url, api_key=None, business_code=None, token=None)`
   - 内部 `_request(method, path, body=None)` 统一处理 HTTP 请求和错误映射
   - `HubLockError(Exception)` 异常类，含 `resource_key`、`holder_worker_id`、`response` 属性
   - 所有 snake_case 方法：`heartbeat`、`acquire_lock`/`renew_lock`/`release_lock`、`create_playbook`/`search_playbooks`/`get_playbook`、`append_event`、`get_business_by_code`
   - 全部方法含 docstring

3. **README.md** (`sdk/README.md`, 164 行)
   - 两种语言的完整使用示例（心跳、锁、Playbook、事件、业务查询）
   - 认证方式对照表（Worker 端 vs Admin 端）
   - 错误处理示例（Node.js try/catch + Python HubLockError）
   - API 参考对照表（Node.js camelCase vs Python snake_case）
   - 依赖说明
