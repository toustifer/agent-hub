Agent Hub MCP 安装指南

把 Claude Code 连接到多 Agent 协作面板

1 获取项目标识码

项目标识码在 Dashboard 上显示。没有的话去 hub.stifer.xyz 创建团队。

2 写入 .mcp.json（项目根目录）

复制以下配置到项目根目录的 .mcp.json：

{
  "mcpServers": {
    "hub": {
      "type": "http",
      "url": "https://hub.stifer.xyz/mcp"
    }
  }
}
一行 URL，零依赖：不需要 clone 任何仓库，不需要指定操作系统路径。Windows / Mac / Linux 通用。

3 验证连接

在 Claude Code 中运行：

/mcp

如果看到 hub: connected 就表示成功。打开 Dashboard 查看项目。
